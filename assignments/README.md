# Assignments for Programming for Data Analytics
## Author: Carmine Giardino

**Assignments** folder for the Programming for Data Analytics course

## Setup
Before running any assignments, install the required dependencies:
```bash
pip install -r requirements.txt
```

This will install:
- `requests` - for fetching data from APIs
- `pandas` - for data manipulation and analysis
- `matplotlib` - for creating visualizations

## Overview
This directory contains:
1. Assignment 02: This script fetches bank holiday data from a public API, lists all Northern Ireland bank holidays, and identifies those unique to Northern Ireland.  
To run use the command: `python assignment02-bankholidays.py`
2. Assignment 03: This notebook downloads the data of 1000 people in csv format, extracts the domains from their email addresses, counts the occurrences of each domain, and plots a pie chart to visualize the distribution of email domains.
3. Assignment 05: This notebook does weighted descriptive statistics on population data by analyzing the differences between the sexes by age in Ireland. It is based on the analysis done by in the course "Programming for Data Analysis" link.
4. Assignment 06: This notebook analyzes of data from Knock airport weather station. Plotting temperature trends over time. And then plotting the wind speed trends.  

The notebooks can be run using the Jupyter Notebook.
